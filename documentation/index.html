A) Preparing - top

Make sure you have the following:

  1. PHP >= 7.0.0
  2. PDO PHP Extension
  3. Tokenizer PHP Extension
  4. OpenSSL PHP Extension
  5. XML PHP Extension
  6. Mbstring PHP Extension
  7. GD Library
  8. CURL Library
  9. PHP mod_rewrite enabled

Make sure you have a mysql database already created, and also make sure that the database schema have already been imported.

B) Installation Process - top

First, let us say that in your hosting server, you have current /public_html/ directory, which is accessible publicly via web domain, for example:

/home/hosting-user/public_html/

Now open the /public_html/ directory in your server and upload the script.zip file downloaded from your purshase download page and unzip it.

At this point, you can find a ".env" file in the /public_html/ directory, if not, make sure the option to see hidden files is enabled on your hosting server.

Open the "public_html/.env" file and edit the database information , APP_NAME, APP_URL, and email information.

APP_NAME= fintech
APP_ENV=local
APP_KEY=base64:NY0K0NvBZJKYObAyHI9v90CyQBKFxA8+1rWqETYoAEQ=
APP_DEBUG=true
APP_LOG_LEVEL=debug

DB_CONNECTION=mysql
